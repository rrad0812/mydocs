
# Convert HTML to Reflex

To convert HTML, CSS, or any design into Reflex code, use our AI-powered build tool at Reflex Build.

Simply paste your HTML, CSS, or describe what you want to build, and our AI will generate the corresponding Reflex code for you.

## How to use Reflex Build

- Go to Reflex Build
- Paste your HTML/CSS code or describe your design
- The AI will automatically generate Reflex code
- Copy the generated code into your Reflex application
